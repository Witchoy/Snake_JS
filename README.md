# 🐍 Snake_JS
A simple Snake game made using HTML, CSS, and vanilla JavaScript.

## 🗒️ Features
- Customizable grid size
- Keyboard controls (Arrow keys and ZQSD)
- Adjustable game speed
- Score tracking
